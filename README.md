## 简介

Shanyue Design 是一款基于 Vue 3 和 TypeScript 构建的免费开源海报设计器，技术栈前沿，开箱即用，助您快速搭建创意设计工具。

核心功能：

- 全面支持画布操作（拖拽、缩放、图层管理）
- 直接导入 PSD 文件，无缝衔接设计稿
- 内置二维码、条形码生成工具，满足推广物料需求
- 灵活扩展，适配各类设计场景

适用场景广泛：

- 海报/广告设计、Logo 制作、AI 图片合成
- 二维码推广海报、电商产品图、名片设计
- 节假日海报、社交媒体配图等图片编辑处理

无论是个人创作者还是企业团队，都能借助 Shanyue Design 高效产出专业视觉内容，开源免费，轻松上手。

## 安装使用

```
# 克隆项目

# github
git clone https://github.com/er-shao/ShanyueDesign.git

# 进入项目目录
cd gzm-design

# 安装依赖
npm install

# 启动项目
npm run dev
```
